# Linkit jQuery Plugins

**Linkit** is a simple jQuery plugin to attach links to elements

## Version
0.2.0

## Usage
    $('span').linkIt({
    href: 'http://test.com',
    text: 'this text',
    target: '_blank'
      });
## Vendors
jQuery - [jQuery Site](https://jquery.com)
